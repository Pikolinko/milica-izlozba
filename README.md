# Izložba slika — Mlada slikarka

Ovo je statički sajt (HTML/CSS/JS) spreman za **GitHub Pages**.

## Stranice
- `index.html` — početna (bio + info o izložbi, prostoru, tematici)
- `artist.html` — detaljnije o slikarki
- `location.html` — lokacija + Google mapa (embed)
- `gallery.html` — slike vertikalno + info sa strane

> Sadržaj i slike su **placeholder** (koristi Unsplash "source" linkove), zamenićeš ih svojim materijalom.

## Kako objaviti na GitHub Pages (brzo)
1. Napravi novi repo (npr. `izlozba-site`)
2. Ubaci fajlove iz ovog foldera u repo (root)
3. GitHub → **Settings** → **Pages**
4. `Build and deployment` → `Source`: **Deploy from a branch**
5. `Branch`: `main` / `(root)`
6. Sačuvaj — GitHub će dati URL (tipično `https://korisnik.github.io/izlozba-site/`)

## Izmena sadržaja
- Promeni tekstove direktno u HTML fajlovima
- Zameni URL-ove slika u `index.html`, `artist.html`, `gallery.html`
- Promeni mapu u `location.html` (traži `map_query`/iframe URL)
